The Dataset Keeping things simple

Our group initial idea was to implement a computer vision model that can identify an animal from an image. But we faced some challenges along the way and couldn't really implement the computer vision model. Because of that we are keeping things simple by using the dataset provided in the demo. The dataset is a collection of SMS spam and ham messages from the UK. Learn more about the dataset at UCI repo

The ML Algorithm

For training the model, we are using the MLP (Multilayer perceptron) algorithm. From our research, the MLP was a popular machine learning solution in the 1980s for classification and recognition models. Because our main goal for the model is to categorize spam and ham messages, we chose the MLP algorithm to train our model. Learn our about the algorithm at Wikipedia

The Purpose of the Model What does it do?

The main purpose of the model is to identify whether a message from the input field is a spam or a ham message, and the confidence percentage of the result. Check out our inference page.
